Most deep learning models for image segmentation are trained on fully annotated pixel-wise masks — a process that is expensive, slow, and often impractical in real-world domains like remote sensing, where images are large, complex, and require domain expertise to label accurately.

This project addresses that gap by building a complete weakly supervised segmentation pipeline trained on nothing but sparse point labels — a handful of labeled pixels per class per image, rather than full masks.

At the core of the project is a custom loss function, PartialFocalLoss, implemented from scratch in PyTorch. It is a direct implementation of the Partial Cross-Entropy formulation, which restricts gradient computation exclusively to labeled point locations using a binary supervision mask, completely ignoring unlabeled pixels during training. A focal weighting component further sharpens learning by down-weighting easy, well-classified examples and focusing the model's attention on harder, ambiguous regions — a critical advantage when supervision is sparse and class imbalance is high.

The dataset used is LoveDA Urban, a publicly available remote sensing benchmark covering seven semantic classes across urban aerial imagery: background, building, road, water, barren land, forest, and agriculture. A custom PointLabelSampler class simulates realistic sparse annotation by randomly sampling N pixel locations per class from full ground truth masks, generating the sparse supervision signal used at training time. The model is a UNet with a pretrained ResNet-34 encoder, trained with AdamW, CosineAnnealingLR scheduling, and automatic mixed precision for efficient training on a Tesla T4 GPU.

Two systematic ablation experiments were designed and executed. The first explores the effect of supervision density — varying the number of labeled points per class from 1 to 20 — to understand how annotation budget affects segmentation quality. The second explores the focal loss gamma parameter across five values, investigating how the balance between easy and hard example weighting influences convergence and final performance under weak supervision.

The full pipeline — loss function, point sampler, dataset class, training loop, experiment management, and evaluation — is implemented in a clean, well-documented Jupyter notebook with over 30 unit tests and runs end to end on Google Colab.
